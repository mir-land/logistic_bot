# logistic_bot

Данный бот разработан для удобства логистов

выполняет функцию рассылки по группам в Whatsapp

так же возможна - отложенной рассылки по таймеру
очистка истории списка групп - разово или по графику
а так же получать и делиться статистикой - общее количество отправленных сообщений для рассылки и общее количество групп для рассылки
можно указать кому из пользователей есть доступ к статистике
есть возможность создать свою команду пользователей для получения статистики

для работы приложения необходимо установить две среды разработки до запуска и перезапустить PC
если при запуске программа просит что то установить - то установите из архива - стабильные верси

1 - для поддержки C# - запустите из архива установщик - windowsdesktop-runtime-6.0.36-win-x64.exe
или скачайте здесь - https://dotnet.microsoft.com/en-us/download/dotnet/6.0

2 - для поддержки Node.js - запустите из архива установщик - node-v24.12.0-x64.msi
или скачайте здесь - https://nodejs.org/en/blog/release/v24.12.0

после запуска программа попросит продлить срок использования ключа
для этого нужно перейти в телеграмм бот - https://t.me/Fenix_licence_bot

отправите боту команду запуска - /start
далее отправьте свой ключ из программы боту - в формате FNX-XXX-XXX

бот предложит получить пробный период или оплатить использование через телеграмм звёзды

для привязки программы к телеграмм аккаунту - запросите в программе привязку - программа в логах сообщит код - далее его нужно ввести боте  через нажатие кнопки "добавить устройство"
при отвязке устройства - старый пользователь получит обновление
